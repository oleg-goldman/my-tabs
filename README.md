# Мои табы ⚡

Личная стартовая страница в духе [hello-nav](https://github.com/hello-nav/hello-nav):
плитки с моими ссылками, мгновенный поиск с клавиатуры, тёмная тема, поиск по
истории браузера. Синхронизация между компьютерами — через git, приватность —
через шифрование.

**Сайт:** https://oleg-goldman.github.io/my-tabs/
**Репозиторий:** https://github.com/oleg-goldman/my-tabs

---

## Как устроено

Проект состоит из двух частей, использующих один и тот же код интерфейса:

1. **Сайт на GitHub Pages** — статические HTML/CSS/JS без фреймворков и сборки.
   Работает в любом браузере на любом компьютере.
2. **Chrome-расширение** (`extension/`) — подменяет новую вкладку тем же
   интерфейсом и добавляет то, что сайту недоступно: поиск по истории браузера.
   Ссылки расширение берёт с сайта, поэтому после `git push` они обновляются
   и во всех установленных расширениях — сами расширения трогать не нужно.

### Поток данных

```
links.local.json  ──npm run encrypt──▶  links.enc.json  ──git push──▶  GitHub Pages
 (только локально,   PBKDF2 + AES-GCM     (в репозитории)                    │
  в .gitignore)      + favicon'ы внутрь                        ┌─────────────┴──────────────┐
                                                               ▼                            ▼
                                                       сайт (любой браузер)      расширение (новая вкладка,
                                                                                  + поиск по истории)
```

### Файлы

| Файл | Назначение |
|---|---|
| `index.html`, `style.css`, `app.js` | сайт: интерфейс, расшифровка, поиск |
| `links.local.json` | **ваши ссылки открытым текстом — только на вашем компьютере** (в `.gitignore`) |
| `links.example.json` | пример формата |
| `links.enc.json` | зашифрованные ссылки — единственное, что публикуется |
| `scripts/encrypt.mjs` | шифрует `links.local.json` → `links.enc.json`, встраивает favicon'ы |
| `scripts/decrypt.mjs` | обратная операция (для нового компьютера) |
| `extension/` | Chrome-расширение: `manifest.json`, `newtab.html`, `config.js` + копии `app.js`/`style.css` (обновляются через `npm run build:ext`) |

### Приватность

- В репозитории и на сайте ссылки существуют **только в зашифрованном виде**
  (ключ из пароля через PBKDF2-SHA256, 310 000 итераций; шифрование AES-256-GCM).
  Посторонний, открывший сайт или репозиторий, увидит форму пароля и бинарный блоб.
- Пароль вводится **один раз на каждый браузер** — ключ кэшируется в localStorage,
  дальше страница открывается мгновенно без ввода.
- Favicon'ы сайтов скачиваются один раз при шифровании и встраиваются в блоб
  data-URI: готовая страница **не делает ни одного внешнего запроса** и не
  раскрывает, какими сайтами вы пользуетесь.
- Страница отдаётся с `noindex` и нигде не публикуется, но URL можно считать
  открытым — защита именно в шифровании.

## Ежедневное использование

### Изменить ссылки

```bash
# правите links.local.json, затем:
npm run encrypt                                   # спросит пароль
git add links.enc.json && git commit -m "links" && git push
```

Через ~минуту обновится на всех компьютерах — и на сайте, и в расширениях.

### Формат ссылки

```json
{ "name": "GitHub", "url": "https://github.com", "keywords": ["git", "код"] }
```

- `keywords` — необязательные синонимы для поиска;
- `icon` — необязательный свой URL/data-URI иконки (тогда favicon не скачивается);
- ссылки группируются по категориям: `{ "categories": [ { "name": "...", "links": [...] } ] }`.

### Горячие клавиши

| Клавиша | Действие |
|---|---|
| любая буква или `/` | фокус в поиск |
| `↑ ↓ ← →` | выбор результата (включая секцию «История») |
| `Enter` | открыть выбранное |
| `⌘Enter` / `Ctrl+Enter` | открыть в новой вкладке |
| `Esc` | сбросить поиск |

## Установка на новом компьютере

### Просто пользоваться

Открыть https://oleg-goldman.github.io/my-tabs/ и один раз ввести пароль.

### Расширение «новая вкладка» (+ поиск по истории)

```bash
git clone https://github.com/oleg-goldman/my-tabs.git
```

1. Chrome → `chrome://extensions`
2. Включить **Режим разработчика** (переключатель справа вверху)
3. **Загрузить распакованное расширение** → выбрать папку `my-tabs/extension`
4. Открыть новую вкладку (`Cmd+T`), один раз ввести пароль

При поиске под результатами появляется секция **«История»** — до 8 страниц из
истории браузера (свои ссылки из неё автоматически исключаются, чтобы не
дублировались). Стрелки и `Enter` работают по обеим секциям.

> Chrome при первом открытии спросит, оставить ли новое расширение новой
> вкладкой — подтвердите («Оставить»).

### Редактировать ссылки на новом компьютере

```bash
npm run decrypt        # восстановит links.local.json по паролю
```

## Разработка

- Локальный просмотр сайта: `npm run serve` → http://localhost:4173
- После изменения `app.js`/`style.css` обновить их копии в расширении:
  `npm run build:ext`, закоммитить и на каждом компьютере сделать
  `git pull` + кнопка «Обновить» на `chrome://extensions`.
  (Это нужно только при изменении интерфейса; изменения ссылок доезжают сами.)
- `extension/config.js` — адрес, откуда расширение берёт `links.enc.json`.

## Смена пароля

```bash
npm run encrypt        # ввести новый пароль
git add links.enc.json && git commit -m "rotate password" && git push
```

На каждом браузере страница один раз спросит новый пароль (старый ключ
перестанет подходить автоматически — соль в блобе новая).
