# Мои табы ⚡

Личная стартовая страница в духе [hello-nav](https://github.com/hello-nav/hello-nav):
плитки с вашими ссылками, мгновенный поиск с клавиатуры, тёмная тема.
Синхронизация между компьютерами — через git, приватность — через шифрование:
в репозитории и на сайте лежит только зашифрованный `links.enc.json`,
посторонний увидит лишь форму ввода пароля.

## Как это работает

- `links.local.json` — ваши ссылки открытым текстом. Лежит **только на вашем
  компьютере** (в `.gitignore`), в репозиторий не попадает.
- `npm run encrypt` — шифрует его паролем (PBKDF2 + AES-GCM) в `links.enc.json`
  и заодно скачивает favicon'ы сайтов, встраивая их прямо в файл — готовая
  страница не делает ни одного внешнего запроса.
- Страница при первом открытии спрашивает пароль и запоминает ключ в браузере —
  дальше открывается мгновенно, без ввода.

## Первый запуск

```bash
cp links.example.json links.local.json   # заполните своими ссылками
npm run encrypt                          # спросит пароль
npm run serve                            # локальный просмотр: http://localhost:4173
```

## Публикация на GitHub Pages

```bash
git init && git add -A && git commit -m "my tabs"
gh repo create my-tabs --public --source . --push
```

Затем на GitHub: **Settings → Pages → Deploy from a branch → main / (root)**.
Через минуту страница доступна на `https://<username>.github.io/my-tabs/`.
Поставьте её домашней страницей или новой вкладкой браузера
(в Chrome — расширением типа «New Tab Redirect»).

## Изменить ссылки

```bash
# правите links.local.json, затем:
npm run encrypt
git add links.enc.json && git commit -m "update links" && git push
```

Через минуту обновится на всех компьютерах.

## Новый компьютер

```bash
git clone <repo> && cd my-tabs
npm run decrypt        # восстановит links.local.json по паролю (если хотите редактировать)
```

Для простого использования достаточно открыть сайт и ввести пароль один раз.

## Формат ссылки

```json
{ "name": "GitHub", "url": "https://github.com", "keywords": ["git", "код"] }
```

`keywords` — необязательные синонимы для поиска. Можно добавить свой `icon`
(URL или data URI) — тогда favicon скачиваться не будет.

## Горячие клавиши

| Клавиша | Действие |
|---|---|
| любая буква или `/` | фокус в поиск |
| `↑ ↓ ← →` | выбор результата |
| `Enter` | открыть |
| `⌘Enter` / `Ctrl+Enter` | открыть в новой вкладке |
| `Esc` | сбросить поиск |
